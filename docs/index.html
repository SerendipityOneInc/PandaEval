Blind tests. Real signal.

A Self-Evolving Framework for
Evaluating AI Agent Skills

ZooEval is a self-evolving evaluation engine that blind-tests AI agent skills by running each task twice—with and without the skill—and grading outputs against deterministic assertions and LLM-as-judge rubrics. Two closed feedback loops let the engine sharpen its own methodology and automatically rewrite underperforming skills.
